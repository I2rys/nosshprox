# NoSSHProx
Automatically block IPs that are proxy that made a SSH request to your server.

## Installation
Github:
```
git clone https://github.com/I2rys/nosshprox
```

NpmJS:
```
npm i parallel-park request-async shelljs lodash fs
```

Packages:
```
sudo apt install iptables -y
```

## Usage
```
node index.js
```
